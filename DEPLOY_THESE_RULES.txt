COPY AND PASTE THESE RULES INTO FIREBASE CONSOLE:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // TEMPORARY MIGRATION RULES - ALLOW PUBLIC WRITES
    match /missing_persons/{personId} {
      allow read, write: if true;
    }
    
    // Keep all other collections secure during migration
    match /{document=**} {
      allow read, write: if false;
    }
  }
}

INSTRUCTIONS:
1. Go to: https://console.firebase.google.com/project/save-them-now/firestore/rules
2. Replace ALL existing rules with the rules above
3. Click "Publish"
4. Wait for deployment to complete (usually 1-2 minutes)
5. Then re-run the migration script

After migration is complete, restore the original secure rules from firestore.rules.backup
